## 安装

克隆仓库：
```bash
git clone https://github.com/psdkjoon/ytmd
```
运行安装脚本：
```bash
./install.sh
```
## 使用方法
```bash
ytmd [音乐/播放列表链接 | 链接文本文件]
```
## 卸载

没有卸载功能 —— 这应用太棒了，你舍不得删 :/

开玩笑的，命令如下：
```bash
sudo rm -rf /usr/bin/ytmd
```
