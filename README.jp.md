## インストール

リポジトリをクローン：
```bash
git clone https://github.com/psdkjoon/ytmd
```
インストールスクリプトを実行：
```bash
./install.sh
```
## 使い方
```bash
ytmd [音楽/プレイリストのURL | URLが記載されたテキストファイル]
```
## アンインストール

アンインストール機能はありません — このアプリは素晴らしすぎて削除できません :/

冗談です。こちらがコマンド：
```bash
sudo rm -rf /usr/bin/ytmd
```
