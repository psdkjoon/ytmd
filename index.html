Simple Yet Powerful

ytmd is a command-line tool that makes downloading music from YouTube incredibly easy. No complex setup, no bloated GUI - just pure, simple functionality.

Built entirely in Bash, ytmd leverages the power of yt-dlp to provide fast, reliable downloads with support for single tracks, playlists, and batch downloads.

Getting Started

Installation

Quick Install

Clone the repository and run the install script:

# Clone the repository
git clone https://github.com/psdkjoon/ytmd
# Navigate to directory
cd ytmd
# Run installer
./install.sh

Uninstallation

If you ever want to remove ytmd (but why would you? üòâ):
